# Facebook Clone

## Project Structure

- `src/` - Contains all the source code for the application.
    - `components/` - React components used throughout the app.
    - `pages/` - Contains different pages like Home, Profile, etc.
    - `styles/` - CSS or styled components used for styling.
- `public/` - Static files like images and icons.
- `tests/` - Contains unit and integration tests.
- `package.json` - Project metadata and dependencies.
- `README.md` - This README file explaining the project.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/humayundanish05/Facebook-clone.git
   cd Facebook-clone
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the application**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.

5. **Running tests**:
   ```bash
   npm test
   ```
